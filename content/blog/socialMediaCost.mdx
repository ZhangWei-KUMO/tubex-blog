# 运维成本
短视频直播类APP的分辨率分为360P、720P和1080P三种。一般以720P为基础标准。如果在APP中打开直播，使用了720p的分辨率，每个用户占用资源0.5M宽带资源（当然实际情况中直播应用还有消息推送，送礼，支付等行为，直播画面分辨率、压缩比等区别，实际会消耗更多的数据）。

| 主要视频格式与码率 |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- |
| 分辨率 | 200P | 360P | 720P | 1080i | 1080P |
| 码率 | 256 | 512 | 1024 | 1800 | 3500 |
| 清晰度 | 流程 | 高清 | 超清 |  |  |

直播平台的带宽成本费用通常取带宽峰值月结，当月 10 万最高同时在线人数，每个用户占用资源0.5M，15块钱1M的样子，月结费用 75万人民币，根据直播方需求不同，价格也不一样，质量越好的就越贵对于直播应用来说，服务器最难处理的环节就是视频流量和用户交互等高频率高带宽的场景，用户的行为是难以预测的，经常会出现突发性的暴涨，一般在进行活动的时候，流量可能是平时的几十倍。第二块的主要成本就是CDN成本。直播平台离不开CDN加速支持，离不开CDN和云厂商，解决诸如：不流畅、卡顿、花屏、带宽不够、攻击、用户体验不好等一系列问题。这块成本相对较低大约单个用户在1块钱左右。本案例将一款直播类APP分为四个阶段，根据国内APP月活量为注册用户10%的平均值取值，月活量分别为10万、50万、100万、50万进行运维成本计算。

|  | 单月成本（万元） |  |  |  |
| --- | --- | --- | --- | --- |
| 注册用户数（万） | 100 | 500 | 1000 | 5000 |
| MAU月活人数（万） | 10 | 50 | 100 | 500 |
| 带宽与CDN费用 | 80 | 400 | 800 | 4000 |

# 审核成本
在直播过程中一些主播为了达到吸引眼球，或者推销商品的目的而做出违规的事情。目前行业内主要以机器审核+人工干预两种方式。以阿里云报价为标准5000万次的内容审核一年的费用在3万元。
